BOX MODEL

The Four Layers Every Web Developer Must Know

The CSS box model is the foundation of all web layouts. Each HTML element is a rectangular box composed of four layers: the content area, padding, border, and margin. Understanding how these four layers stack and interact is the first step toward total layout control.

#02
POSITIONING

Static, Relative, Absolute, Fixed & Sticky: Explained

CSS offers five position values. Static is the default — elements flow naturally. Relative moves an element from its normal position while keeping its space. Absolute removes it from flow and places it relative to its nearest positioned ancestor. Fixed pins it to the viewport. Sticky acts like relative until a scroll threshold, then becomes fixed.

BORDER STYLES

All 8 Border Styles & How to Mix Them Per Side

CSS gives you eight border styles: solid, dotted, dashed, double, ridge, groove, inset, and outset. You can apply different styles to each side using the shorthand border-style: solid groove dashed dotted — top, right, bottom, left — just like margin and padding shorthand.
